1. Project Overview

This food-ordering chatbot transforms a static hotel website into a smart conversational platform. Leveraging Dialogflow for NLP and FastAPI + MySQL for the backend, customers can simply type “New order,” add items to their cart, check status, or cancel—all in plain English.

2. Problem Statement

Traditional restaurant websites force users to navigate static menus and forms. This creates friction:

  • Users must search through long lists of items.
  • Every change in menu requires a code update.
  • Tracking orders often relies on delayed human responses.

As a result, customers experience frustration and restaurants shoulder unnecessary support overhead.

3. Solution

We scoped an MVP by selecting two high-impact, high-feasibility features:

  1. Order Placement: Show live menu, add/remove items, confirm order.
  2. Order Tracking: Fetch real-time status by order ID.

Using a Feasibility-Impact Analysis, we avoided low-value additions (e.g. payment gateway) for the first release, ensuring rapid deployment and user feedback.

4. Architecture & Data Flow

1. Dialogflow ES handles intent detection & context.
2. FastAPI routes requests—modular handlers for “new_order”, “track_order”, etc.
3. MySQL stores menu and orders; stored procedures ensure atomic transactions.

5. Key Features

  • Conversational UX: Fallback intents gracefully handle out-of-scope queries.
  • Dynamic Menu: Menu items read from a database table—no redeploy required to update.
  • Order Cart: In-memory session store until checkout or cancellation.
  • Atomic Operations: Add/remove via stored procedures with full rollback on errors.
  • Order Tracking: Pull current status (“Received”, “In Kitchen”, etc.) by order ID.

7. Deployment & Security

• Local Dev: `uvicorn main:app --reload --port 8000` + Cloudflare Tunnel.
• Production: Deployed on Render with SSL certs; originally used Aiven MySQL, now supports both hosted and local DB.
• CI/CD: GitHub Actions automates tests & deployment.

8. Future Enhancements

  • Payment Integration: Add Stripe/PayPal for in-chat checkout.
  • Multi-lingual Support: Use Dialogflow’s language models for Spanish, French, etc.
  • Analytics Dashboard: Track user behavior, popular items, peak hours.
  • Voice Interface: Integrate Dialogflow Voice for phone-based orders.
