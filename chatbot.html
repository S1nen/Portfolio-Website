AI-Powered Food Ordering Chatbot

This project is an end-to-end AI-powered chatbot built to streamline food ordering using natural language. Developed using Dialogflow for conversational intelligence and FastAPI as the backend, the chatbot is capable of handling order placement, tracking, and cancellation seamlessly. The backend connects to a secure Aiven-hosted MySQL database to persist and retrieve order data, ensuring real-time communication with users.

Users can interact with the chatbot directly through a conversational interface, available both on the deployed web application and integrated into the project portfolio. Hosted using Render and secured with SSL certificates, this chatbot simulates a complete digital ordering experience suitable for any restaurant or café. The system is designed to reduce manual overhead, speed up ordering, and improve customer satisfaction.

Live Demo: Try the Chatbot Live

Project Components:

  1. Dialogflow for NLP & Intent Detection
  2. FastAPI backend with modular route handling
  3. MySQL database integration (hosted on Aiven)
  4. Webhook communication with Dialogflow
  5. Deployment on Render + Frontend Integration

Features & Tech Stack

  • Natural conversation with fallback & context support
  • Real-time order insertion, tracking, and cancellation
  • Secure SSL-based MySQL connection using certificate authentication
  • Frontend with Dialogflow Messenger embedded chatbot
  • Cloudflare Tunnels for local testing before deployment
