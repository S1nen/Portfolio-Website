Sentiment Analysis System

This project focuses on building a sentiment analysis system that evaluates user reviews (e.g., product reviews, movie reviews) and classifies them into positive, negative, or neutral sentiments. The core of the system is an XGBoost machine learning model trained on preprocessed Amazon review data. The frontend interface allows users to input a review and receive real-time predictions, making it an effective tool for understanding customer sentiment.

The model leverages CountVectorizer for feature extraction and MinMaxScaler for normalization. The project integrates seamlessly with a Flask web application and uses a clean, interactive frontend built with HTML and CSS. The final system is capable of receiving user input via a form and displaying the sentiment prediction instantly on the same page. This practical use case demonstrates the end-to-end lifecycle of a machine learning deployment project from data preprocessing to model integration and deployment.

Project Components:

  1. Data Preprocessing & Feature Engineering
  2. Sentiment Classification using XGBoost
  3. Flask Web App Integration & UI Deployment

Data Analysis & Model Details

  • Data Source: Amazon product reviews dataset
  • Text Vectorization: CountVectorizer (max_features=4000)
  • Feature Scaling: MinMaxScaler for numeric stability
  • Model: XGBoost classifier with hyperparameter tuning
  • Accuracy: High F1-score with balanced precision and recall

Model Evaluation Steps:
  1. Train-test split: 80/30 ratio
  2. Confusion matrix, precision, recall, F1-score computed
  3. Tested live predictions via the frontend interface

Web App Functionality

  • Users can enter text reviews directly into a form on the web interface.
  • On submission, the system returns the predicted sentiment label.
  • Clear separation of backend (Flask), ML logic, and frontend code for maintainability.
  • Supports real-time interaction, providing immediate results without page reloads.
